Additional Lab 4: Advanced Blocking Pages
================================================================================

In this module, you will the existing L3 Outbound Topology that was created in previous module with a policy that implements the **Advanced Blocking Pages** service extension for managing a dynamic blocking pages. The service can either be deployed as a static blocking page at the end of a blocking traffic policy, or can apply iRule logic to dynamically inject blocking page content. 

|

.. note::
   This lab builds upon the existing **BIG-IP SSLO L3 Outbound Proxy** configuration from the previous labs:

   - ``Lab 1: Outbound Transparent Forward Proxy with User Coaching``
   - ``Lab 2: DNS-over-HTTPS (DoH) Guardian``
