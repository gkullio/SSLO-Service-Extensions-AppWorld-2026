Additional Lab 3: SaaS Tenant Isolation
================================================================================

In this module, you will the existing L3 Outbound Topology that was created in previous module with a policy that implements **SaaS Tenant Isolation** for managing tenant restrictions for several SaaS applications in a corporate environment.
